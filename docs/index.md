# AssTrack Documentation

AssTrack is an asset tracking platform for fleet assets, live location feeds, geofences, alerts, and bridge-based integrations.

## Start Here

- [Repository README](https://github.com/JerrettDavis/AssTrack#readme)
- [Integration guide](integrations.md)
- [OpenAPI contract](openapi/asstrack.v1.json)

## Operational Areas

- Run the full local stack with Aspire.
- Configure provider feeds and bridge gateway routes from the in-app Integrations page.
- Attach multiple trackers to a single tracked asset.
- Monitor live map positions, geofence events, and bridge-generated observations.
